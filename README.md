# Python_Expense_Tracker

A command-line expense tracking application built with Python for recording, categorizing, and analyzing personal expenses.

## Features

* Add expenses
* View expense history
* Categorize expenses
* Track total spending
* Store expense records
* Simple command-line interface

## Technologies Used

* Python
* File Handling
* Command Line Interface (CLI)

## How to Run

1. Clone the repository.
2. Open a terminal in the project folder.
3. Run the application:

```bash
python expense_tracker.py
```
